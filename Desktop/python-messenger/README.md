# Python Messenger

Простой мессенджер на Python для ПК с GUI (Tkinter) и сервером Flask + SocketIO.

## Как запустить

1. Установить зависимости:
   pip install -r requirements.txt

2. Запустить сервер:
   python server/main.py

3. Вставить URL сервера в client/client.py и запустить клиент:
   python client/client.py
